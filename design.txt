
## 1️⃣ **Theme & Style**

* **Primary Theme:** Dark (professional, control room feel)
* **Secondary Theme:** Light (optional toggle for user preference)
* **Style Inspiration:** CCTV monitoring, cybersecurity dashboards, map-heavy emergency systems

**Colors:**

| Element              | Color Code      | Usage                      |
| -------------------- | --------------- | -------------------------- |
| Background           | `#0B132B`       | Main dashboard background  |
| Secondary BG         | `#1C2541`       | Panels, cards              |
| Primary UI (buttons) | `#3A86FF`       | Buttons, highlights        |
| Alerts: High Risk    | `#FF4D4F`       | Red zones, SOS alerts      |
| Alerts: Medium       | `#FFD60A`       | Medium crime zones         |
| Alerts: Safe         | `#2ECC71`       | Safe zones                 |
| Alerts: Accident     | `#00B4D8`       | Traffic / accident overlay |
| Text                 | White `#FFFFFF` | Main text                  |
| Text Light Gray      | `#B0B3B8`       | Subtext, secondary info    |

## 2️⃣ **Layout Structure**

### **Sidebar (Left)**

* Full height, fixed
* Icons + labels:

  * Dashboard
  * Map
  * FIR Entry
  * Patrol
  * Alerts
* Hover effect: Highlighted background (`#3A86FF`) + icon glow

### **Top Bar**

* Fixed at top
* Components:

  * User Profile (top-right)
  * Notifications (bell icon)
  * Optional search bar
* Subtle shadow for depth

### **Main Content Area**

* **Dashboard View:**

  * Cards with crime stats, top hotspots
  * Quick SOS alerts panel
  * KPI metrics (daily, weekly, monthly)

* **Map View:**

  * Full width map
  * Color-coded zones (red/yellow/green/blue)
  * Hover popup → crime type + risk score
  * Smooth zoom & transition animations
  * Active alerts → glowing/red blinking

---

## 3️⃣ **UI Components**

* **Cards:** For stats → clean, rounded corners, subtle shadow
* **Buttons:** Primary blue (`#3A86FF`) with hover glow
* **Popups / Tooltips:** For map hover → soft dark background + light text
* **Alerts / Notifications:** Flashing or glowing badges
* **Tabs / Filters:** Top of map → smooth animation, toggle buttons

---

## 4️⃣ **UX Considerations**

* **High contrast:** Dark background + bright alert colors → improves readability
* **Interactive map:** Hover for more info, clickable zones
* **Alerts:** Immediate visual feedback (blinking / glowing)
* **Responsive design:** Works on desktop and tablet
* **Optional Light Theme:** Same components, lighter colors, softer contrast

---

## 5️⃣ **Map Design Tips**

* Danger zones → glowing red
* Medium zones → amber overlay
* Safe zones → soft green
* Accidents → blue overlay
* Opacity 0.6–0.8 → realistic
* Hover → tooltip with details

---

## 6️⃣ **Extra UI Tricks for Judges**

* Glow effect on active alerts
* Hover popups with stats for map zones
* Smooth animations for map transitions
* Clean, uncluttered panels with rounded cards
